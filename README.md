# 📸 电子翻页相册

一个真实的3D翻页相册，支持手机和电脑观看。

## ✨ 特性

- 📱 手机优化 - 触摸滑动翻页
- 🎨 真实3D翻页效果 - 纸张弯曲动画
- 🎵 翻页音效 - 真实的翻书声音
- ✋ 可拖拽控制 - 拖到一半可停止或返回
- 📖 Canvas渲染 - 高性能流畅动画

## 🚀 在线预览

[查看相册](https://dianzixiangce.vercel.app)

## 📝 使用说明

### 电脑端操作
- 点击页面左侧/右侧切换页面
- 或使用键盘方向键 ← →
- 拖拽页面可控制翻页进度

### 手机端操作
- 左右滑动翻页
- 拖拽可控制翻页进度
- 拖到一半可停止或返回

## 🛠️ 技术栈

- StPageFlip.js - 专业翻书库
- Canvas - 高性能渲染
- 响应式设计 - 自适应各种屏幕
